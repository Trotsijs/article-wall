### Installation:

1. Clone or Download the project.
2. Run:
````
composer install
````
3. Navigate to `/public` directory:
````php
cd .\public\
````
4. Start a server from the terminal
```php
php -S localhost:8000
```


5. Navigate to http://localhost:8000 to see the site.
